# Neo-Chat

```
https://neo-chat-ashy.vercel.app
```

## Skills

- Next.js 14(App router), React 18
- TypeScript
- React Context API
  - User 정보를 전역 관리
- Supabase
  - Realtime
  - Table
- Tailwind CSS
  - 반응형
- Git / Github
