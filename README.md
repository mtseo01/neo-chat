# Neo-Chat

실시간 익명 채팅 웹 애플리케이션입니다. Smith 요원이 되어 다른 이와 대화해 보세요!

demo:
https://neo-chat-ashy.vercel.app

## Motivation

실시간 채팅을 구현하고 싶었고, 최근에 영화 매트릭스를 너무 재밌게 봐서 매트릭스 테마로 만들어 보았습니다.

## Skills

- Next.js 14(App router), React 18
- TypeScript
- React Context API
  - User 정보를 전역 관리
- Supabase
  - Realtime
  - Table
- Tailwind CSS
  - 반응형
- Git / Github
